Publier sur Google Sites

📋 Étape 1 — Copier le code HTML de l'examen

Ce fichier HTML est autonome (self-contained). Il peut être hébergé sur n'importe quel serveur ou intégré dans Google Sites.

✅ Code copié dans le presse-papier !
🌐 Étape 2 — Intégration dans Google Sites
Méthode recommandée :
1. Ouvrez votre page Google Sites
2. Cliquez sur Insérer → Intégrer
3. Collez l'URL du fichier HTML hébergé (ex: GitHub Pages, Netlify, Google Drive public)
4. Ajustez la hauteur à 900px minimum

Hébergement gratuit recommandé :
• GitHub Pages (github.com → Settings → Pages)
• Netlify Drop (app.netlify.com/drop) — glissez-déposez le fichier HTML
• Google Drive avec partage public
📊 Étape 3 — Recevoir les notes (Google Forms optionnel)
Les résultats sont déjà enregistrés automatiquement dans cet outil (onglet Résultats).

Pour envoyer aussi les notes vers Google Forms :
1. Créez un formulaire Google avec les champs : Nom, Session, Score (%), Points, Résultat
2. Récupérez l'URL de pré-remplissage de votre formulaire
3. Configurez-la ci-dessous :
URL Google Forms (pré-remplissage) — optionnel
ID champ Nom
ID champ Score %
